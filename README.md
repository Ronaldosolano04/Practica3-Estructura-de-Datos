# Sistema de Gestión de Colecciones en C#

Este proyecto consiste en el desarrollo de una aplicación de consola en **C# (.NET 6 o superior)** que implementa el uso de las principales estructuras de datos genéricas de la biblioteca `System.Collections.Generic`: **List**, **Queue** y **Stack**. El sistema está organizado mediante un menú principal que permite acceder a tres módulos independientes. El primer módulo administra una **lista de estudiantes**, permitiendo agregar nuevos estudiantes con su nombre y calificación, eliminar estudiantes por nombre, mostrar todos los registros y validar que no existan duplicados. El segundo módulo implementa una **cola de atención al cliente (FIFO)**, donde los clientes son atendidos en el mismo orden en que llegan, ofreciendo opciones para agregar clientes, atender al siguiente cliente en espera y visualizar la cola completa, además de evitar registros repetidos. El tercer módulo utiliza una **pila de tareas pendientes (LIFO)** para gestionar actividades, permitiendo agregar nuevas tareas, marcar como completada la tarea más reciente y mostrar todas las tareas pendientes. El programa incorpora validaciones para evitar operaciones sobre colecciones vacías, controlar entradas inválidas y mejorar la experiencia del usuario mediante menús interactivos y mensajes informativos. El objetivo principal del proyecto es demostrar el funcionamiento práctico de las estructuras de datos más utilizadas en C#, aplicando buenas prácticas de programación, organización del código, reutilización de métodos y una interfaz de consola sencilla, intuitiva y fácil de utilizar.
